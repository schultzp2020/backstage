# @backstage/cli-module-oxlint

CLI module that provides oxlint-powered lint commands for the Backstage CLI.

## Commands

| Command        | Description                                   |
| :------------- | :-------------------------------------------- |
| `package lint` | Lint a package using oxlint                   |
| `repo lint`    | Lint all packages in the project using oxlint |
